Your DOS games folder

This folder has been created by Boxer as a place to keep your DOS games: Boxer will put new games here when it imports them.

Boxer has stocked this folder with free DOS games for you to try: just double-click on a game to start it up. Check out the Boxer website for other sample games, and tips on where to find more DOS gaming goodness.

You can rename this folder or move it wherever you like – Boxer will still find it. Feel free to throw away this document once you’re done reading.
