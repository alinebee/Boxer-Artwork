The DOS Utilities folder

This folder has been created by Boxer as a handy place to put your DOS utilities. Anything you put in this folder will be available to each DOS game and application you run. You can access its contents from the DOS prompt at Y:\UTILS, and you can run any program in here from anywhere in DOS: just by typing the name of the program.

You can also rename this folder or move it wherever you like—Boxer will still find it. Feel free to throw away this document once you’re done reading.
